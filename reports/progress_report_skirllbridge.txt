ALUR PENGERJAAN - ORANG 1 (AI & Advanced ML Engineer)

1. Merging & Preprocessing: Menggabungkan dataset mentah JobStreet (~298 MB) dan membersihkan nama karir agar tidak duplikat.
2. Skill Extraction: Mengekstrak keahlian wajib dari lowongan kerja menggunakan kamus skill berbasis pencocokan Regex (\b) dan normalisasi sinonim (seperti menggabungkan java script ke javascript).
3. Vektorisasi (Embedding): Menggunakan model Sentence-Transformers "all-MiniLM-L6-v2" untuk membuat representasi vektor karir (Mean Pooling) dari 20 lowongan teratas secara offline.
4. Lazy Loading: Model Sentence-Transformer hanya dimuat ke memori saat pengguna melakukan pencarian secara real-time untuk menghemat RAM.
5. Query Alignment: Mengubah input skill pengguna menjadi format kalimat: "The candidate is proficient and has skills in: [skills]."
6. Semantic Vector Search: Menghitung kemiripan kosinus (Cosine Similarity) antara vektor kueri pengguna dengan vektor database karir.
7. Semantic Scoring & Calibration: Mengkalibrasi nilai kesamaan semantik mentah ke skala persentase 0-100% yang ramah pengguna.
8. Penentuan Kelayakan: Memfilter karir tanpa data skill wajib, menguji tumpang tindih keahlian, membatalkan rekomendasi jika tidak ada kecocokan sama sekali (kembali kosong dengan info peringatan), serta memberikan deskripsi peran Indonesia dan label kelayakan standar (Sangat Cocok >= 75, Cocok >= 50, Kurang Cocok < 50).
